# Minesweeper MorphDOM

This is a simple implementation of minesweeper for the morphdom using:

- hyperscript-helpers
- MorphDOM
- mostjs
- ramdajs

## Development Setup

```
npm install
npm start
```

open browser to http://localhost:3000
